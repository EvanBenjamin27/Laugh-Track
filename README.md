# Laughtrack.live

Laughtrack is a free online web game to play with friends where players anonymously respond to an unfinished phrase or missing word in a sentence and people vote on the funniest one

## How to run:

The entire page works statically and all the functions run in javascript on the host and client sides. It is as simple as adding the release to a service like Github pages or Cloudflare workers/pages to host the site and using firebase realtime database messaging for everything else. So it is very easy to get your own version running.
